# Proyecto de Digitalización Operativa - Espacios Públicos (Rocha)

##  Descripción
Proyecto de análisis y diseño de un sistema de digitalización operativa para el Área de Espacios Públicos de la Intendencia Departamental de Rocha.

Este trabajo es de carácter **conceptual**, basado en el relevamiento de necesidades reales detectadas en procesos administrativos y operativos del área. No se encuentra implementado.

##  Objetivo
Proponer una solución modular y escalable que permita mejorar la gestión interna mediante la digitalización de procesos, la centralización de información y la optimización operativa.

##  Módulos del sistema propuesto
- Portal institucional de espacios públicos
- Gestión de palmeras (picudo rojo)
- Registro de palmeras en predios privados
- Sistema de memorandos (memos)
- Gestión de vivero
- Gestión de obrador e insumos

##  Estado del proyecto
Proyecto conceptual (fase de análisis y diseño).
